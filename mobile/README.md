# Mobile App (Flutter)

This directory now contains a custom Flutter app scaffold.

Current implementation:

- App theme and brand tokens
- Guest-first auth flow UI
- Session restore/logout flow
- Customers home screen
- API client wiring using Dio

Suggested MVP screens:

- Splash / Onboarding
- Guest start + Login / Register
- Customers list
- Add customer
- Customer details + measurement history
- Add measurement
- Plan/Upgrade screen
- Backup/Restore status

Run commands:

- `& "c:\mobile-app-projects\flutter\bin\flutter.bat" pub get`
- `& "c:\mobile-app-projects\flutter\bin\flutter.bat" run -d chrome`
